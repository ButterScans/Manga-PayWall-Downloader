# Manga PayWall Downloader
Uma extensão simples pra donos de scans conseguirem raws em sites fácies/difícies que contem(ou não) PayWall(capítulos pagos). 

# Como usar?
Em cada site suportado haverá um botão chamado: "Salvar páginas". Apenas clique e baixe. Se você estiver em uma versão desatualizada, a extensão avisará você.

# Sites suportados (até o momento):
1- https://comic-fuz.com/ | 2- https://cycomi.com/ | 3- https://takecomic.jp/ | 4- https://manga-mee.jp/ | 5- https://championcross.jp/ | 6- https://firecross.jp/ | 7- https://bookwalker.jp + https://viewer.bookwalker.jp/ | 8- https://ynjn.jp/ | 9- https://manga-one.com/
